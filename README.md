# CLOSERS_Install_Helper
클로저스를 빠르게 설치할 수 있게 도와주는 도구입니다.

## 설치 또는 실행
### 다운로드
[릴리즈 탭](https://github.com/Korokk3/CLOSERS_Install_Helper/releases)에서 최신 버전을 다운로드 받을 수 있습니다.

### 소스 코드로 실행
```bash
py -3 -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
py -3 app.py
```


## 안내
본 프로그램 이용시 발생하는 모든 문제에 대해 책임지지 않습니다.

현재는 다운로드 속도가 개선된 것으로 확인되니 본 도구를 사용하지 않아도 쾌적하게 설치할 수 있을 것 같습니다.
