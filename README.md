## Getting Started
### Installation
```
git clone https://github.com/Korokk3/CLOSERS_Install_Helper.git
py -3 -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
py -3 app.py
```

`requests` 말고는 다 기본 모듈인듯 하니 굳이 가상환경 만들 필요는 없을듯
